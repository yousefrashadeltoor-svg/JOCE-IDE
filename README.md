# JOCE IDE Project

Offline-first multi-file project version of JOCE IDE.

## What is included
- `index.html` - main app shell
- `css/style.css` - application styling
- `js/app.js` - original app logic extracted from the single-file build
- `js/enhancements.js` - added project-level enhancements
- `backgrounds/` - 10 generated PNG wallpapers
- `data/themes/` - 10 theme preset JSON files
- `data/locales/` - localization seed files
- `data/changelog.json` - update history
- `extensions/` - 10 sample extension packages
- `manifest.webmanifest` - PWA manifest
- `sw.js` - offline cache service worker
- `icons/` - app icons

## Safe developer center
The project includes a local-only developer center that shows app health, update history, project stats, extension counts, and storage usage. It does not include cross-device personal tracking.
